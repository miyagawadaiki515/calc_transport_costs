ローカルに配置後、以下を実行してブラウザで操作できるようになります。
```
①依存関係をインストール
$ npm install

②scriptsを確認
$ npm run

③開発サーバーを起動
$ npm run dev

④ブラウザでアクセス
http://localhost:5173/
```
